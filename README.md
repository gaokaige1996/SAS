### 3. Generate Data, 
```
data oranges; 
     input variety $ flavor texture looks; 
     total = flavor + texture + looks; 
     label total = 'Sum'; 
CARDS; 
navel 9 8 6 
temple 7 7 7 
valencia 8 9 9 
mandarin 5 7 8  
;
```
 "input" tells SAS which variable we generated, $ means character, total is generated by other three variable. <br>
